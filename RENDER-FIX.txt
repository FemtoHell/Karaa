# ========================================
# FIX LỖI RENDER: "Could not read package.json"
# ========================================

❌ LỖI HIỆN TẠI:
   Render đang chạy ở root directory (/)
   Nhưng backend code nằm trong folder /backend/
   → Không tìm thấy package.json

✅ GIẢI PHÁP: Có 2 cách

# ========================================
# CÁCH 1: CẬP NHẬT RENDER SETTINGS (ĐƠN GIẢN NHẤT)
# ========================================

1. Vào Render Dashboard: https://dashboard.render.com

2. Chọn service "resume-builder-api"

3. Vào tab "Settings"

4. Tìm phần "Build & Deploy"

5. Cập nhật các giá trị sau:

   📁 Root Directory:
      backend

   🔨 Build Command:
      npm install

   ▶️ Start Command:
      npm start

6. Kéo xuống dưới, click "Save Changes"

7. Render sẽ tự động redeploy

8. Đợi 2-3 phút → DONE! ✅

# ========================================
# CÁCH 2: DÙNG render.yaml (TỰ ĐỘNG)
# ========================================

Tôi đã tạo file render.yaml ở root directory.

1. Push file render.yaml lên GitHub:
   git add render.yaml RENDER-FIX.txt
   git commit -m "Add Render config"
   git push origin main

2. Vào Render Dashboard

3. Xóa service cũ (nếu cần)

4. Tạo service mới:
   - Click "New" → "Web Service"
   - Connect GitHub repo: FemtoHell/BeThanh
   - Render sẽ tự động phát hiện render.yaml
   - Click "Apply"

5. Add environment variables từ file backend-env-vars.txt

6. Deploy → DONE! ✅

# ========================================
# KHUYẾN NGHỊ
# ========================================

👉 Dùng CÁCH 1 vì nhanh và đơn giản hơn.

Chỉ cần:
- Root Directory: backend
- Build Command: npm install
- Start Command: npm start

Sau đó add environment variables là xong!
